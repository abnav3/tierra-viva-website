# Casa Mazunte Tierra Viva 🌴

Sitio web oficial de **Casa Mazunte Tierra Viva**, un hotel boutique de estilo mexicano rodeado de manglar en Mazunte, Oaxaca.

## Características

- Página web estática, 100% HTML + CSS (sin frameworks ni build tools)
- Diseño responsivo con modo claro/oscuro
- SEO optimizado (Open Graph, Schema.org JSON-LD, hreflang multilenguaje)
- Galería de fotos de las habitaciones y áreas comunes
- Enlaces directos a reservas por WhatsApp y Airbnb
- Premios y certificaciones destacadas

## Estructura

```
.
├── index.html              # Página principal

```
